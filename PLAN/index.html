    # 작업(Task) 설명(Description) 우선순위
    1.1 mkdocs.yml 설정 Material Theme, 한글 검색, 네비게이션 구성 ⭐⭐⭐
    1.2 index.md 작성 랜딩 페이지 — 프로젝트 소개 및 퀵링크 ⭐⭐⭐
